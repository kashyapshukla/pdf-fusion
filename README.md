# PDF Fusion

A secure, client-side web application to merge multiple PDF files into one. Built with React, Vite, and pdf-lib.

## Features
- **Secure Handling**: All processing happens in your browser. No files are uploaded to any server.
- **Drag & Drop**: Easy interface to upload multiple files.
- **Reordering**: Drag files to reorder them before merging.
- **Password Support**: Merge password-protected PDFs by providing the password securely.
- **Premium UI**: Dark mode with smooth animations and glassmorphism design.

## How to Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
